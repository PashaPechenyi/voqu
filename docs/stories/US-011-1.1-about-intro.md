# US-011-1.1: About Page - Intro Section

## Story

**As a** visitor
**I want to** read an introduction about what Voqu is
**So that** I understand the platform's purpose and target audience

---

## Description

Create the introduction section for the About page that explains what Voqu is and who it's designed for. This sets the context for the rest of the About page content.

---

## Acceptance Criteria

- [ ] Create IntroSection component at `apps/web/src/pages/public/AboutPage/sections/IntroSection.tsx`
- [ ] Display section title: "Про Voqu"
- [ ] Display two paragraphs of introductory text (in Ukrainian)
- [ ] Title centered, paragraphs left-aligned
- [ ] Paragraphs have increased font size for readability
- [ ] Text color uses text.secondary for body content
- [ ] Container has narrower max-width for comfortable reading

---
