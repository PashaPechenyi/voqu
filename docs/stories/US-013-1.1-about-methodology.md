# US-013-1.1: About Page - Methodology Section

## Story

**As a** visitor
**I want to** understand how lessons are structured
**So that** I know what to expect from the learning experience

---

## Description

Create a section that explains the lesson methodology using a visual timeline. Shows the four main components of each lesson in sequential order.

---

## Acceptance Criteria

- [ ] Create MethodologySection component at `apps/web/src/pages/public/AboutPage/sections/MethodologySection.tsx`
- [ ] Display section title: "Як побудовані уроки"
- [ ] Display subtitle: "Кожен урок складається з кількох компонентів для ефективного засвоєння"
- [ ] Show 4 steps in a Timeline component (from @mui/lab)
- [ ] Each step has: icon in colored dot, title, description in Paper card
- [ ] Timeline alternates left/right on desktop
- [ ] White background

---

## Technical Notes

**Libraries:**
- `@mui/material` - Box, Container, Typography, Paper
- `@mui/lab` - Timeline, TimelineItem, TimelineSeparator, TimelineConnector, TimelineContent, TimelineDot
- `@mui/icons-material` - Abc, MenuBook, Headphones, Quiz

**File location:** `apps/web/src/pages/public/AboutPage/sections/MethodologySection.tsx`

**Required installation:**
```bash
npm install @mui/lab
```

**Timeline configuration:**
- position="alternate" (left/right alternating)
- TimelineDot color="primary", padding p={1}
- TimelineConnector between items (not after last)
- TimelineContent contains Paper

**Paper styling:**
- elevation={2}
- padding: p={2}
- borderRadius: 2
