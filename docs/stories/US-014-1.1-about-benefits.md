# US-014-1.1: About Page - Benefits Section

## Story

**As a** visitor
**I want to** understand the advantages of learning with Voqu
**So that** I can decide if this platform is right for me

---

## Description

Create a section that highlights four key benefits of using Voqu for learning English. Uses a two-column grid with icon + text layout for each benefit.

---

## Acceptance Criteria

- [ ] Create BenefitsSection component at `apps/web/src/pages/public/AboutPage/sections/BenefitsSection.tsx`
- [ ] Display section title: "Переваги навчання з Voqu"
- [ ] Show 4 benefits in a 2-column responsive grid
- [ ] Each benefit has: icon in colored box, title, description
- [ ] Desktop: 2 columns, Mobile: 1 column
- [ ] Background: grey.50

---

| Icon | Title (UA) | Description (UA) |

## Technical Notes

**Libraries:**
- `@mui/material` - Box, Container, Typography, Grid, Stack
- `@mui/icons-material` - Schedule, AccountTree, TrendingUp, CardGiftcard

**File location:** `apps/web/src/pages/public/AboutPage/sections/BenefitsSection.tsx`

**Icon box styling:**
- padding: p={1.5}
- borderRadius: 2
- bgcolor: primary.main
- color: white
- flexShrink: 0 (prevent shrinking)

**Grid configuration:**
- Container spacing: 4
- xs: 12 (full width)
- sm: 6 (2 columns)

**Layout per benefit:**
- Stack direction="row" with spacing={3}
- Icon box on left
- Text content on right (Box)
