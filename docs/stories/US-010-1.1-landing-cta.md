# US-010-1.1: Landing Page - CTA Section

## Story

**As a** visitor
**I want to** see a final call-to-action at the bottom of the landing page
**So that** I'm reminded to sign up after seeing all the features

---

## Description

Create a prominent call-to-action section at the bottom of the landing page with primary (signup) and secondary (learn more) buttons. This section uses the primary brand color as background for visual impact.

---

## Acceptance Criteria

- [ ] Create CTASection component at `apps/web/src/pages/public/LandingPage/sections/CTASection.tsx`
- [ ] Display headline: "Готовий розпочати?"
- [ ] Display subtext: "Приєднуйся до тисяч українців, які вивчають англійську з Voqu"
- [ ] Show primary button: "Створити акаунт" → navigates to /auth/signup
- [ ] Show secondary button: "Дізнатися більше" → navigates to /about
- [ ] Background: primary.main color from theme
- [ ] Text color: white (primary.contrastText)
- [ ] Desktop: buttons side by side
- [ ] Mobile: buttons stacked vertically

## UI/UX Requirements

- Section should feel like a strong conclusion to the page
- Content centered, textAlign center
- Container maxWidth: md (focused width)
- Stack spacing={4} for vertical rhythm between elements
- Headline: h3 variant, fontWeight 600
- Subtext: h6 variant, opacity 0.9 for subtle hierarchy
- Buttons should be equally prominent but primary slightly more

---

## Notes

- This mirrors the hero CTA but with more urgency after user has seen content
- White buttons on colored background create strong contrast
- Secondary button allows hesitant users to learn more before committing
