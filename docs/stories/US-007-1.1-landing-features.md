# US-007-1.1: Landing Page - Features Section

## Story

**As a** visitor
**I want to** see the main features of the application
**So that** I understand what benefits I'll get from using Voqu

---

## Description

Create a features section that displays four key benefits of using Voqu in a responsive card grid. Each feature has an icon, title, and brief description.

---

## Acceptance Criteria

- [ ] Create FeaturesSection component at `apps/web/src/pages/public/LandingPage/sections/FeaturesSection.tsx`
- [ ] Display section title: "Чому обирають Voqu?"
- [ ] Show 4 feature cards in a responsive grid
- [ ] Each card displays: MUI icon, title (Ukrainian), description (Ukrainian)
- [ ] Cards have consistent height within each row
- [ ] Cards have border styling (no shadow, 1px border)
- [ ] Icons use primary theme color
- [ ] Responsive grid: 1 column (xs), 2 columns (sm), 4 columns (md)

---

## Feature Content

| Icon | Title | Description |
|------|-------|-------------|
| School | Структуровані рівні | Від A1 до C2 за стандартом CEFR |
| MenuBook | Різноманітні уроки | Словник, граматика, читання, аудіювання |

## Technical Notes

**Libraries:**
- `@mui/material` - Box, Container, Grid, Card, CardContent, Typography, Stack
- `@mui/icons-material` - School, MenuBook, Timeline, Bookmark

**File location:** `apps/web/src/pages/public/LandingPage/sections/FeaturesSection.tsx`

**Card styling:**
- elevation={0} (no shadow)
- border: 1px solid, borderColor: 'divider'
- borderRadius: 2 (16px with theme)
- padding: p={2}
- textAlign: center
- height: 100%

**Grid breakpoints:**
- xs: 12 (full width)
- sm: 6 (2 per row)
- md: 3 (4 per row)

**Icon size:** fontSize: 48 (48px)

---

## UI/UX Requirements

- Section padding: py={6} on xs, py={10} on md
- Title margin bottom: mb={6}
- Grid spacing: 4 (32px gap)
- Cards should feel light and airy
- Icons should be prominent but not overwhelming
- Use Stack inside CardContent for consistent spacing
