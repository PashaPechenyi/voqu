# US-008-1.1: Landing Page - Levels Preview Section

## Story

**As a** visitor
**I want to** see a visual representation of the learning levels
**So that** I understand the structured progression path available

---

## Description

Create a section that visualizes the CEFR levels (A1 through C2) using MUI's Stepper component. This shows visitors the learning journey from beginner to proficiency level.

---

## Acceptance Criteria

- [ ] Create LevelsPreviewSection component at `apps/web/src/pages/public/LandingPage/sections/LevelsPreviewSection.tsx`
- [ ] Display section title: "Твій шлях до вільної англійської"
- [ ] Display subtitle: "Навчання побудоване за міжнародним стандартом CEFR"
- [ ] Show 6 CEFR levels in a Stepper component
- [ ] Each level shows: colored circle with code (A1, A2...), name, Ukrainian description
- [ ] Desktop: horizontal stepper with labels below
- [ ] Mobile: vertical stepper
- [ ] Background: grey.50
- [ ] Each level has a unique color

## Technical Notes

**Libraries:**
- `@mui/material` - Box, Container, Typography, Stepper, Step, StepLabel, useMediaQuery, useTheme

**File location:** `apps/web/src/pages/public/LandingPage/sections/LevelsPreviewSection.tsx`

**Stepper configuration:**
- orientation: horizontal (md+) / vertical (below md)
- alternativeLabel: true on horizontal, false on vertical
- activeStep: -1 (no step active, all shown equally)

**Custom StepIcon:**
- Circular box (40x40px, borderRadius 50%)
- Background color from level data
- White text with level code
- fontWeight: 700
- fontSize: 0.875rem

**Responsive detection:**
```typescript
const isMobile = useMediaQuery(theme.breakpoints.down('md'));
```
