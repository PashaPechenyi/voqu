# US-006-1.1: Landing Page - Hero Section

## Story

**As a** visitor
**I want to** see an engaging hero section when I land on the homepage
**So that** I immediately understand what Voqu offers and feel motivated to sign up

---

## Description

Create the hero section for the landing page that displays a compelling headline, subheadline, and call-to-action button. This is the first thing visitors see and should clearly communicate the app's value proposition.

---

## Acceptance Criteria

- [ ] Create HeroSection component at `apps/web/src/pages/public/LandingPage/sections/HeroSection.tsx`
- [ ] Display headline: "Вивчай англійську крок за кроком"
- [ ] Display subheadline: "Структуровані уроки від початківця до просунутого рівня для українців"
- [ ] Show CTA button: "Почати безкоштовно" that navigates to /auth/signup
- [ ] Apply gradient background (light grey to white, top to bottom)
- [ ] Content centered horizontally and vertically aligned
- [ ] Text centered on all screen sizes
- [ ] Subheadline has a max-width (600px) for readability
- [ ] Responsive padding: xs=64px, md=96px vertical

---

## Technical Notes

**Libraries:**
- `@mui/material` - Box, Container, Typography, Button, Stack
- `react-router-dom` - useNavigate

**File location:** `apps/web/src/pages/public/LandingPage/sections/HeroSection.tsx`

**Typography variants:**
- Headline: h1 variant, fontWeight 700
- Subheadline: h2 variant, fontWeight 400, color text.secondary

**Responsive font sizes:**
| Element | xs | md |
|---------|----|----|
| Headline | 2.5rem | 3.5rem |
| Subheadline | 1.25rem | 1.5rem |

**Button styling:**
- variant="contained"
- size="large"
- padding: px={4}, py={1.5}
- fontSize: 1.1rem

**Background gradient:**
```css
background: linear-gradient(180deg, #f5f7fa 0%, #ffffff 100%)
```

---

## UI/UX Requirements

- Hero should feel spacious and not cramped
- CTA button should stand out as the primary action
- Use Stack with spacing={4} for vertical rhythm
- Container maxWidth="lg" for consistent width
- Section uses semantic `<Box component="section">`

---

## Notes

- This is typically the highest-impact section for conversion
- Keep the message simple and focused
- The gradient background creates visual separation from header
