# US-016-1.1: About Page Assembly

## Story

**As a** developer
**I want to** assemble all about page sections into a complete page
**So that** visitors can learn about Voqu with proper SEO meta tags

---

## Description

Create the main AboutPage component that combines all section components (Intro, CEFR, Methodology, Benefits) and adds SEO meta tags using react-helmet-async.

---

## Acceptance Criteria

- [ ] Create folder structure at `apps/web/src/pages/public/AboutPage/`
- [ ] Create `AboutPage.tsx` as the main page component
- [ ] Create `index.tsx` for re-export
- [ ] Import and render all 4 sections in correct order
- [ ] Add Helmet component with SEO meta tags
- [ ] Page title: "Про нас — Voqu"
- [ ] Meta description: "Дізнайтеся про методологію Voqu та рівні CEFR. Структурований підхід до вивчення англійської для українців."
- [ ] Wrap sections in semantic main element

---

## Technical Notes

**Libraries:**
- `@mui/material` - Box
- `react-helmet-async` - Helmet

**File structure:**
```
apps/web/src/pages/public/AboutPage/
├── index.tsx
├── AboutPage.tsx
└── sections/
    ├── IntroSection.tsx
    ├── CEFRSection.tsx
    ├── MethodologySection.tsx
    └── BenefitsSection.tsx
```

**Section order:**
1. IntroSection
2. CEFRSection
3. MethodologySection
4. BenefitsSection

**Helmet usage:**
```tsx
<Helmet>
  <title>Page Title</title>
  <meta name="description" content="Description" />
</Helmet>
```
