# US-009-1.1: Landing Page - Lesson Preview Section

## Story

**As a** visitor
**I want to** see what a real lesson looks like
**So that** I can understand the learning format before signing up

---

## Description

Create a section that displays a static mockup of a vocabulary lesson. This gives visitors a preview of the learning experience without requiring backend data.

---

## Acceptance Criteria

- [ ] Create LessonPreviewSection component at `apps/web/src/pages/public/LandingPage/sections/LessonPreviewSection.tsx`
- [ ] Create lessonPreviewData.ts file with static mockup data
- [ ] Display section title: "Як виглядає урок"
- [ ] Display subtitle: "Приклад уроку зі словником рівня A1"
- [ ] Show a Card containing a sample vocabulary lesson
- [ ] Card header shows: level chip (A1), lesson title, subtitle
- [ ] Card content shows 3 vocabulary items
- [ ] Each vocabulary item shows: word, translation, example sentence, part of speech chip
- [ ] Items separated by dividers
- [ ] Card has elevation and rounded corners

---

**Lesson info:**

## Technical Notes

**Libraries:**
- `@mui/material` - Box, Container, Typography, Card, CardContent, CardHeader, List, ListItem, Chip, Divider, Stack

**File locations:**
- `apps/web/src/pages/public/LandingPage/sections/LessonPreviewSection.tsx`
- `apps/web/src/pages/public/LandingPage/sections/lessonPreviewData.ts`

**Card styling:**
- elevation={3}
- maxWidth: 500px
- mx: 'auto' (centered)
- borderRadius: 3 (24px)

**CardHeader styling:**
- bgcolor: grey.50
- Title contains level Chip + Typography in a Stack

**Vocabulary item layout:**
- Use ListItem with flexDirection: column, alignItems: flex-start
- Word and part of speech on same row (space-between)
- Word: h6 variant, color primary
- Part of speech: Chip, size small, variant outlined
- Translation: body1, fontWeight 500
- Example: body2, color text.secondary, fontStyle italic, quoted

---

## UI/UX Requirements

- Section padding: py={6} on xs, py={10} on md
- Container maxWidth: md (narrower for focus)
- Title centered, mb={2}
- Subtitle centered, color text.secondary, mb={6}
- List disablePadding
- ListItem padding: py={2}
- Divider between items (not before first)
