# US-012-1.1: About Page - CEFR Section

## Story

**As a** visitor
**I want to** understand the CEFR language proficiency levels in detail
**So that** I can assess my current level and understand my learning path

---

## Description

Create a section that explains the CEFR (Common European Framework of Reference) levels using expandable accordion panels. Each level shows its description and specific skills users will acquire.

---

## Acceptance Criteria

- [ ] Create CEFRSection component at `apps/web/src/pages/public/AboutPage/sections/CEFRSection.tsx`
- [ ] Display section title: "Рівні володіння мовою (CEFR)"
- [ ] Display subtitle explaining CEFR: "Загальноєвропейські рекомендації з мовної освіти — міжнародний стандарт для опису рівня володіння мовою"
- [ ] Show 6 accordion panels (A1-C2), each expandable independently
- [ ] Each panel header shows: colored level chip, English name, Ukrainian name
- [ ] Each panel content shows: description paragraph, "Що ви зможете:" heading, skills list with checkmarks
- [ ] Background: grey.50
- [ ] Accordions have subtle elevation
