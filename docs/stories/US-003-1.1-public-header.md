# US-003-1.1: Public Header

## Story

**As a** visitor
**I want to** see a navigation header on every page
**So that** I can easily navigate between pages and access login/signup

---

## Description

Create a responsive header component for public pages that displays navigation links on desktop and a hamburger menu with drawer on mobile devices.

---

## Acceptance Criteria

### Desktop (md breakpoint and above)
- [ ] Display Logo component on the left side, clicking navigates to home
- [ ] Show navigation buttons: "Головна" (/) and "Про нас" (/about)
- [ ] Show "Увійти" button linking to /auth/login
- [ ] Show "Реєстрація" button (contained/filled style) linking to /auth/signup
- [ ] Navigation items should be horizontally aligned on the right

### Mobile (below md breakpoint)
- [ ] Display Logo on the left
- [ ] Show hamburger menu icon on the right
- [ ] Clicking hamburger opens a Drawer from the right side
- [ ] Drawer contains list items for all navigation: Головна, Про нас, Увійти
- [ ] Drawer contains "Реєстрація" as a contained button at the bottom
- [ ] Clicking any navigation item closes the drawer and navigates

### General
- [ ] Create component at `apps/web/src/components/layout/PublicHeader.tsx`
- [ ] Header should be sticky (position="sticky")
- [ ] Header has white background with subtle elevation (elevation={1})
- [ ] Use Container with maxWidth="lg" for content width
- [ ] All text is in Ukrainian

---

## Technical Notes

**Libraries:**
- `@mui/material` - AppBar, Toolbar, Container, Button, IconButton, Drawer, List components
- `@mui/icons-material` - Menu icon
- `react-router-dom` - Link, useNavigate

**File location:** `apps/web/src/components/layout/PublicHeader.tsx`

**Navigation items data:**
```typescript
const navItems = [
  { label: 'Головна', path: '/' },
  { label: 'Про нас', path: '/about' },
];
```

**Responsive breakpoint:** Use `useMediaQuery` with `theme.breakpoints.down('md')` for mobile detection

---

## UI/UX Requirements

- Header should feel clean and uncluttered
- Mobile drawer width: 250px
- Navigation buttons use "inherit" color, auth buttons differentiate (Увійти: inherit, Реєстрація: contained)
- Smooth drawer animation (MUI default)
- Logo should be a clickable link to home page

---

## Notes

- Auth routes (/auth/login, /auth/signup) don't exist yet - navigation should still work
- Drawer state managed with useState
- Consider using Link component from react-router-dom for nav items to avoid full page reloads
