# US-004-1.1: Public Footer

## Story

**As a** visitor
**I want to** see a footer with useful links and copyright information
**So that** I can access additional pages and know who owns the site

---

## Description

Create a footer component for public pages that displays navigation links and copyright text. The footer should appear at the bottom of every public page.

---

## Acceptance Criteria

- [ ] Create component at `apps/web/src/components/layout/PublicFooter.tsx`
- [ ] Display copyright text: "© {current year} Voqu. Всі права захищено."
- [ ] Show footer links: "Про нас" (/about), "Контакти" (/contact), "Політика конфіденційності" (/privacy)
- [ ] Links should have hover underline effect
- [ ] Use grey background (grey.100 from theme)
- [ ] Desktop: copyright on left, links on right in a row
- [ ] Mobile: stack vertically with items centered
- [ ] Use Container with maxWidth="lg" for content width
- [ ] Footer automatically sticks to bottom when content is short (mt: 'auto')

---

## Technical Notes

**Libraries:**
- `@mui/material` - Box, Container, Typography, Link, Stack
- `react-router-dom` - Link component

**File location:** `apps/web/src/components/layout/PublicFooter.tsx`

**Footer links data:**
```typescript
const footerLinks = [
  { label: 'Про нас', path: '/about' },
  { label: 'Контакти', path: '/contact' },
  { label: 'Політика конфіденційності', path: '/privacy' },
];
```

**Styling:**
- Padding: py={4} (32px vertical)
- Background: grey.100
- Text color: text.secondary
- Link underline: "hover"

---

## UI/UX Requirements

- Footer should feel minimal and not distract from content
- Year should be dynamically calculated using `new Date().getFullYear()`
- Links should be spaced with gap/spacing of 3 (24px)
- Use MUI Link component with react-router Link for internal navigation
- Responsive stack: row on sm and up, column on xs

---

## Notes

- /contact and /privacy pages don't exist yet - links should still work
- Footer is part of the layout, not individual pages
- Keep styling consistent with overall clean aesthetic
