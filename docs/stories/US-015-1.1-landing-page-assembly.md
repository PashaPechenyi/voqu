# US-015-1.1: Landing Page Assembly

## Story

**As a** developer
**I want to** assemble all landing page sections into a complete page
**So that** visitors can see the full landing page with SEO meta tags

---

## Description

Create the main LandingPage component that combines all section components (Hero, Features, Levels Preview, Lesson Preview, CTA) and adds SEO meta tags using react-helmet-async.

---

## Acceptance Criteria

- [ ] Create folder structure at `apps/web/src/pages/public/LandingPage/`
- [ ] Create `LandingPage.tsx` as the main page component
- [ ] Create `index.tsx` for re-export
- [ ] Import and render all 5 sections in correct order
- [ ] Add Helmet component with SEO meta tags
- [ ] Page title: "Voqu - Вивчай англійську крок за кроком"
- [ ] Meta description: "Безкоштовна платформа для вивчення англійської мови для українців. Структуровані уроки від A1 до C2."
- [ ] Wrap sections in semantic main element

---

## Technical Notes

**Libraries:**
- `@mui/material` - Box
- `react-helmet-async` - Helmet

**File structure:**
```
apps/web/src/pages/public/LandingPage/
├── index.tsx
├── LandingPage.tsx
└── sections/
    ├── HeroSection.tsx
    ├── FeaturesSection.tsx
    ├── LevelsPreviewSection.tsx
    ├── LessonPreviewSection.tsx
    ├── lessonPreviewData.ts
    └── CTASection.tsx
```

**Section order:**
1. HeroSection
2. FeaturesSection
3. LevelsPreviewSection
4. LessonPreviewSection
5. CTASection

**Helmet usage:**
```tsx
<Helmet>
  <title>Page Title</title>
  <meta name="description" content="Description" />
</Helmet>
```
